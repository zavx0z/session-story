# Spec: publish package

Публикационный пакет — это финальный набор текстов и монтажных заметок.

## Структура

```text
gpt-handoff/publish/
  ru-script.md
  en-script.md
  ru-blog-news.md
  en-blog-news.md
  video-description.md
  editing-plan.md
  keyframe-usage.md
```

## RU script

Русская дикторка. Должна быть человеческой, без канцелярита и без пересказа каждого клика.

## EN script

Английская версия. Не дословный перевод, а нормальная адаптация.

## Blog news

Короткая новость:

- что сделали;
- зачем это важно;
- что видно в видео;
- какой следующий шаг.

## Video description

Описание для публикации в медиа/видеохостинге.

## Editing plan

План монтажа:

- где держать экран;
- где ускорить;
- где можно вырезать;
- какие кадры использовать как visual proof;
- где голос должен объяснить смысл.

## Keyframe usage

Пояснение, какие ключевые кадры доказывают какие этапы истории.
