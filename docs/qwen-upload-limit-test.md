# Проверка лимита загрузки изображений в Qwen

Цель: понять, сколько изображений Qwen Chat в браузере стабильно принимает за одно сообщение без потери качества анализа.

## Ручной протокол

Открой Qwen Chat в браузере.

Подготовь тестовые папки:

```text
upload-limit-test/
  04-images/
  08-images/
  12-images/
  16-images/
  20-images/
```

Проверь загрузку:

1. 4 изображения
2. 8 изображений
3. 12 изображений
4. 16 изображений
5. 20 изображений

Для каждого теста проверь:

- файлы успешно загружаются;
- Qwen видит все изображения;
- ответ ссылается на все кадры;
- ответ не превращается в общую воду;
- чат не зависает;
- браузер остаётся стабильным.

Фиксируй результат так:

```text
4 изображения: ok
8 изображений: ok
12 изображений: ok
16 изображений: деградация
20 изображений: не работает
```

## Рекомендация

Используй максимальное стабильное число изображений.

Если отдельные изображения загружаются плохо, используй contact sheet.
